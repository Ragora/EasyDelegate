Detailed Description

A type that can represent any delegate type, but it cannot be invoked without casting to a delegate type that knows the proper function signature.

This delegate type is useful in the event you wish to pass around delegates regardless of the type in a system where the function signature is resolved via other means for type casting and invocation.
